# Portfolio Profesional - Angela Alvarez

## Data Analysis & QA Specialist



Este proyecto es una **Landing Page** profesional desarrollada como trabajo final para el curso de Desarrollo Frontend en **Ada ITW**. El sitio está diseñado bajo una estética tecnológica "Dark Mode" y optimizado para dispositivos móviles.


##  Características del Proyecto

- **Estructura Semántica**: Uso de etiquetas HTML5 (`header`, `section`, `article`, `blockquote`) para mejorar el SEO y la lectura del código.

- **Diseño Responsivo**: Adaptabilidad total a móviles, tablets y escritorio mediante CSS Grid y Flexbox.

- **Seguridad & MFA**: Sección dedicada a la concientización sobre ciberseguridad, con una tarjeta informativa sobre Autenticación de Doble Factor.

- **Accesibilidad (A11y)**: Implementación de atributos `aria-hidden` en iconos decorativos y contraste de color validado para una mejor experiencia de usuario.



## Uso de Inteligencia Artificial

En el desarrollo de este proyecto, se integró el uso de **IA Generativa** para:

- **Optimización de Layout**: Asistencia en el centrado complejo de elementos y alineación simétrica del formulario de contacto mediante Flexbox.


## Tecnologías Utilizadas

- HTML5 / CSS3 Puro

- JavaScript (ES6+)

- Font Awesome (Iconografía)

- Formspree (Gestión de correos)

- GitHub Pages (Deploy)
